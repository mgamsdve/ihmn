# IHMN — Refonte de la plateforme web

## Présentation

Ce projet consiste à repenser et reconstruire la présence digitale de l’Institut d’Hygiène et de Médecine Naturelle (IHMN).

Le site actuel fournit les informations essentielles mais ne reflète pas pleinement la qualité, l’expérience et la réputation de l’institut. L’objectif est de créer une plateforme moderne, claire et évolutive qui mette mieux en valeur les formations et facilite la découverte de l’école pour les futurs étudiants.

Ce projet s’inspire de la proposition stratégique reçue par l’IHMN, tout en étant adapté à une architecture technique moderne et à une mise en œuvre simplifiée.

---

# Objectifs du projet

Les principaux objectifs de cette refonte sont :

- moderniser la présence digitale de l’IHMN
- améliorer la visibilité sur les moteurs de recherche (SEO)
- clarifier la présentation des formations
- améliorer le parcours d’information des futurs étudiants
- permettre à l’équipe de l’IHMN de modifier facilement le contenu
- créer une base technique solide pour les évolutions futures

---

# Architecture du projet

Le nouveau site sera construit avec une **architecture headless moderne**.

## Frontend

Une application web moderne optimisée pour :

- la performance
- le référencement naturel
- l’accessibilité
- l’affichage mobile (responsive)

## Gestion de contenu

Le contenu sera géré via un **Headless CMS (Sanity)** permettant à l’équipe de l’IHMN de :

- modifier les pages
- publier des articles de blog
- gérer la FAQ
- ajouter des témoignages
- mettre à jour les informations sur les formations

---

# Phase 1 — Structure digitale et architecture de contenu

Avant le développement, il est nécessaire de définir la structure du futur site.

Cette phase consiste à :

- analyser le contenu du site actuel
- identifier les pages principales
- définir le parcours d’information des visiteurs
- identifier les opportunités SEO

## Livrables

- arborescence du futur site
- définition des pages principales
- organisation du contenu
- identification des mots-clés SEO
- suggestions de sujets pour le blog

L’objectif est de construire une structure claire à la fois pour les utilisateurs et pour les moteurs de recherche.

---

# Phase 2 — Refonte et développement du site

Cette phase consiste à concevoir et développer la nouvelle plateforme web.

Le nouveau site devra :

- mieux présenter les formations
- valoriser l’expertise de l’institut
- guider les visiteurs vers la prise de contact
- améliorer la lisibilité et la navigation

## Pages principales

Le site comprendra notamment :

- Accueil
- Présentation de l’institut
- Philosophie / vision
- Programmes et formations
- Modalités et admissions
- Débouchés
- Témoignages
- FAQ
- Contact
- Blog

## Objectifs techniques

Le site sera conçu avec les priorités suivantes :

- rapidité de chargement
- design responsive
- structure optimisée pour le SEO
- gestion du contenu via CMS
- architecture évolutive

---

# Fondations SEO

Le site sera construit avec des bases SEO solides.

Cela inclut :

- titres de pages optimisés
- meta descriptions
- hiérarchie claire des titres (H1, H2, H3)
- URLs propres
- stratégie de maillage interne
- données structurées (schema.org)
- génération d’un sitemap
- optimisation des images

Le blog servira de support pour améliorer la visibilité à long terme.

Les sujets d’articles seront proposés mais la rédaction sera réalisée par l’équipe de l’IHMN.

---

# Stratégie de contenu

Le site combinera :

- pages informatives optimisées
- contenu pédagogique via le blog
- explications détaillées des formations
- témoignages d’étudiants

L’objectif est de renforcer la crédibilité de l’institut et d’accompagner les visiteurs dans leur réflexion.

---

# Email et capture de contacts (à définir)

Le site pourra intégrer des mécanismes permettant aux visiteurs de laisser leurs coordonnées :

- formulaire de contact
- inscription à une newsletter
- téléchargement de documents informatifs

Ces éléments permettront éventuellement de maintenir un lien avec les futurs étudiants.

Les modalités exactes seront définies avec l’IHMN.

---

# Évolutions possibles

La plateforme pourra ensuite accueillir d’autres fonctionnalités :

- séquences d’emails automatisées
- promotion d’événements
- campagnes digitales ciblées
- intégrations avec les réseaux sociaux

Ces éléments pourront être ajoutés progressivement.

---

# Principes du projet

La refonte du site repose sur plusieurs principes :

- simplicité
- performance
- maintenabilité
- autonomie pour l’équipe de l’IHMN
- évolutivité à long terme

L’objectif n’est pas seulement de refaire un site, mais de créer une base digitale solide pour le développement futur de l’institut.
