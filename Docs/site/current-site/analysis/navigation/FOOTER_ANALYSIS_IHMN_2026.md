# Analyse du pied de page (Footer) — IHMN

## Description

Le pied de page du site IHMN est présent **de manière identique sur toutes les pages du site**.

Il contient des informations institutionnelles, des coordonnées et quelques éléments de navigation secondaire.

Le footer joue principalement un rôle :

- informatif
- institutionnel
- de contact

plutôt qu’un rôle de navigation.

---

# 1. Structure du footer

Le footer est organisé en **trois blocs principaux**.

## Bloc 1 — Lieu des cours

Informations sur le lieu où se déroulent les cours.

Contenu :

Rue de Bomel 154  
5000 Namur  
Belgique

Observation :

Cette information est utile pour les étudiants mais pourrait être plus clairement associée à la section **formations**.

---

## Bloc 2 — Siège social & secrétariat

Adresse administrative de l’école.

Contenu :

Rue sur la Fontaine 92  
4000 Liège  
Belgique

Observation :

Cette adresse correspond au siège administratif de l’IHMN.

---

## Bloc 3 — Contact

Informations de contact direct.

Contenu :

Email : info@ihmn.be  
Téléphone : +32 (0)4 343 52 86  
Téléphone : +32 (0)491 714 447

Observation :

Les coordonnées sont claires mais pourraient être enrichies avec :

- un lien vers la page contact
- un bouton de prise de contact.

---

# 2. Éléments institutionnels

Le footer contient également plusieurs éléments institutionnels.

### Logo World Naturopathic Federation

Le logo indique que l’IHMN est :

Membre éducatif de la **World Naturopathic Federation (WNF)**.

Observation :

Cet élément renforce la crédibilité de l’école.

---

### Slogan

" La naturopathie, une passion, un art, un métier "

Ce slogan est affiché dans le footer et rappelle la mission de l’école.

---

# 3. Réseaux sociaux

Le footer contient un **lien vers Facebook**.

Observation :

La présence sociale est très limitée.

Aucun lien vers :

- Instagram
- LinkedIn
- YouTube

---

# 4. Navigation secondaire

Le footer contient un seul lien de navigation :

Mentions légales  
https://ihmn.be/mentions-legales

Observation :

Le footer ne propose pas de navigation secondaire complète.

Dans de nombreux sites, le footer inclut également :

- FAQ
- Contact
- Formations
- Politique de confidentialité
- Conditions générales

---

# 5. Cohérence avec le reste du site

Le footer est :

- cohérent graphiquement
- identique sur toutes les pages
- simple et lisible

Cependant il ne contribue pas réellement à la navigation du site.

---

# 6. Analyse UX

Points positifs :

- structure claire
- informations essentielles présentes
- lisibilité correcte

Points perfectibles :

- manque de navigation secondaire
- peu de liens utiles
- présence sociale limitée
- absence de liens vers les pages importantes.

---

# 7. Recommandations pour la refonte

Pour améliorer le footer dans la future version du site, il serait pertinent de :

### Ajouter une navigation secondaire

Exemple :

Formations  
L’école  
FAQ  
Thérapeutes  
Contact

---

### Ajouter les informations légales complètes

Politique de confidentialité  
Conditions générales  
Mentions légales

---

### Améliorer la présence sociale

Ajouter :

Facebook  
Instagram  
LinkedIn

---

### Ajouter un accès rapide aux formations

Exemple :

Naturopathie  
Hygiène de vie et nutrition  
Drainage lymphatique manuel

---

# Conclusion

Le footer actuel est fonctionnel mais reste essentiellement informatif.

Il pourrait être amélioré afin de :

- renforcer la navigation du site
- améliorer le référencement
- faciliter l’accès aux pages importantes.

Lors de la refonte du site, le footer pourra devenir une **zone de navigation secondaire structurante**.
