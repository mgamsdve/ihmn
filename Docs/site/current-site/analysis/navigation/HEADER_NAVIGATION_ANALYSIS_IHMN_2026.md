# Analyse du header et de la navigation principale — IHMN

## Description

Le header du site IHMN est identique sur toutes les pages du site.

Il contient :

- le logo de l'école
- le menu principal de navigation
- un accès Facebook
- une icône de recherche

Ce header constitue l'élément principal de navigation du site.

---

# 1. Structure du header

Le header est organisé en trois zones principales :

| Zone | Contenu |
|-----|-----|
| Gauche | Logo IHMN |
| Centre | Menu principal |
| Droite | Facebook + recherche |

---

# 2. Logo

Le logo IHMN est situé à gauche du header.

Fonction :

- lien vers la page d'accueil.

URL cible :

https://ihmn.be/

Observation :

Comportement standard conforme aux conventions web.

---

# 3. Menu principal

Le menu principal contient les entrées suivantes.

| Item | URL |
|-----|-----|
| Accueil | / |
| Formations | /formations |
| IHMN | /ihmn |

---

# 4. Sous-menu Formations

Le menu **Formations** comporte un menu déroulant.

Sous-pages accessibles :

| Item | URL |
|-----|-----|
| Naturopathie | /formations/naturopathie |
| Hygiène de vie et nutrition | /formations/hygiene-de-vie-et-nutrition |
| Drainage lymphatique manuel | /formations/drainage-lymphatique-manuel |
| Formations hors cursus | /formations/formations-hors-cursus |
| Inscriptions & tarifs | /formations/inscriptions-tarifs |
| Examens & certificats | /formations/examens-certificats |

Observation :

Le menu regroupe toutes les pages liées aux formations.

Cependant :

- l'ordre n'est pas totalement logique
- les pages administratives (inscriptions, examens) sont mélangées aux formations.

# 6. Autres éléments du header

## Icône Facebook

Lien vers la page Facebook de l'IHMN.

Observation :

Seul réseau social présent dans le header.

---

## Icône de recherche

Présence d'une icône de recherche dans le header.

Observation :

La recherche ne semble pas être un élément central du site.

---

# 7. Analyse UX de la navigation

Points positifs :

- navigation simple
- nombre limité d'items
- hiérarchie claire
- accès direct aux formations.

Points perfectibles :

- certaines pages administratives sont mélangées aux formations
- absence de lien direct vers certaines pages importantes
- absence de regroupement logique des contenus pédagogiques.

---

# 8. Cohérence de la navigation

La navigation correspond globalement à la structure réelle du site.

Cependant certains contenus ne sont pas accessibles directement depuis le menu :

- modules de formation
- contenus pédagogiques détaillés
- certaines pages internes.

---

# 9. Analyse SEO

La structure actuelle est relativement simple mais présente plusieurs limites SEO.

Problèmes observés :

- certains contenus importants n'ont pas d'URL propre
- les modules de formation sont affichés en modales
- la navigation n'expose pas toute la structure pédagogique.

---

# 10. Recommandations pour la refonte

Pour améliorer la navigation lors de la refonte du site, plusieurs améliorations pourraient être envisagées.

### Séparer les formations et l'administration

Exemple :

Formations
- naturopathie
- hygiène de vie et nutrition
- drainage lymphatique manuel
- formations hors cursus

Admissions
- inscriptions
- tarifs
- organisation des cours

---

### Améliorer l'accès aux programmes

Ajouter dans les pages formations :

- programme
- modules
- examens
- débouchés.

---

### Améliorer la navigation SEO

Créer des pages dédiées pour :

- les modules
- les années de formation
- les contenus pédagogiques.

---

# Conclusion

La navigation principale du site IHMN est simple et claire, mais elle expose uniquement les pages principales.

Une restructuration pourrait améliorer :

- la compréhension du site
- la navigation entre les contenus
- le référencement naturel.
