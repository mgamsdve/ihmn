# Structure des formations

Synthèse établie uniquement à partir des fichiers d’inventaire fournis.

- Cours de nutrition 1
- Techniques respiratoires

### Année 2

- Approche naturopathique des métabolismes
- Biophysique médicale 1
- Déontologie
- Cours d’hygiène vitale 3
- Cours d’immunologie
- Cours de nutrithérapie 1
- Nutrition 2
- Programmation neuro-linguistique (PNL) 2
- Psychomorphologie
- Cours psychophysiologie

### Année 3

- Anatomie et physiologie 3
- Cours de Clinique 1
- Consultation 1
- Iridologie

- Cours psychophysiologie

- Cours de Clinique 2

Remarques structurelles :

- Organisation clairement séquencée sur 4 années.
- Certains modules de l’année 2 ont une fiche détaillée incohérente ou incomplète dans l’inventaire source : `Nutrition 2`, `Programmation neuro-linguistique (PNL) 2`, `Psychomorphologie`.
- Le module `Analyses biologiques` est présent mais sa fiche est incomplète dans l’inventaire source.

## Formation Drainage Lymphatique Manuel

URL : `/formations/drainage-lymphatique-manuel`

Structure pédagogique : formation courte en module unique.

### Module unique

- Drainage lymphatique manuel (DLM)

Remarques structurelles :

- La formation est structurée autour d’un seul module.
- L’inventaire distingue explicitement une partie théorique et une partie pratique au sein du module.

## Formations Hors Cursus

URL : `/formations/formations-hors-cursus`

Structure pédagogique : modules indépendants.

### Modules

- Le RGPD : Je gère !
- Argile : remède ancestral
- Atelier Lacto-fermentation
- Formation au ‘BGM’ (Bindegewebs Massage)
- Atelier Lacto-fermentation
- Cycle féminin : le comprendre et l’équilibrer naturellement
- Gemmothérapie
- Pharmacologie
- Les plantes médicinales grandeur nature
- Créez votre projet professionnel et développez votre communication

Remarques structurelles :

- Les formations sont proposées comme des séminaires ou journées indépendantes, sans progression par années.
- L’intitulé `Atelier Lacto-fermentation` apparaît deux fois dans l’inventaire source, avec deux formats distincts.
- La fiche de `Créez votre projet professionnel et développez votre communication` comporte une incohérence de titre dans la popup source, mais le contenu correspond bien au module indiqué sur la carte.
