# Inventaire des documents PDF — IHMN (2026)

## Description

Analyse des documents PDF présents sur le site de l'IHMN afin d'identifier :

- les documents actuellement publiés
- leur rôle dans le site
- leur état (actuel ou potentiellement obsolète)
- les recommandations pour une future migration ou refonte du site

Cette analyse vise à déterminer quels documents doivent être :

- conservés
- mis à jour
- supprimés
- remplacés par des pages web.

# 2. Analyse des documents

## Contrat d'inscription

Document utilisé pour formaliser l'inscription des étudiants à la formation.

Contenu typique :

- informations personnelles
- engagement contractuel
- conditions d'inscription
- signature

Observation :

Ce document doit rester téléchargeable en PDF car il nécessite une signature.

Recommandation :

Conserver le PDF, mais vérifier qu'une **nouvelle version soit publiée chaque année académique**.

---

## Tarifs de la formation

Document présentant :

- les tarifs des formations
- les modalités de paiement
- les éventuelles conditions particulières

Observation :

Les tarifs évoluent régulièrement.

Recommandation :

Deux options possibles :

- maintenir le PDF avec mise à jour annuelle
- ou transformer cette information en **page web dynamique plus facile à maintenir**.

---

## Formulaire de demande de bourse

Document utilisé pour introduire une demande de bourse ou d'aide financière.

Observation :

Document administratif nécessitant probablement un téléchargement et une signature.

Recommandation :

Conserver le PDF.

À long terme, il pourrait être remplacé par :

- un formulaire en ligne.

---

## Mentions légales

Le site propose actuellement les mentions légales sous forme de PDF.

Observation :

Ce type d'information est généralement présenté directement sous forme de page web pour :

- améliorer le référencement
- faciliter la consultation
- permettre les mises à jour rapides.

Recommandation :

Remplacer ce PDF par une **page HTML dédiée sur le site**.

Les PDF ne sont pas nécessairement optimisés pour :

- la lecture mobile
- l'accessibilité (lecteurs d'écran).

# 5. Recommandations pour la refonte

Dans le cadre d'une future refonte du site, il est recommandé de :

### conserver en PDF

- contrat d'inscription
- formulaire de bourse

### transformer en pages web

- tarifs des formations
- mentions légales

Cela permettrait :

- une meilleure maintenance
- une meilleure accessibilité
- un meilleur référencement.

---

# 6. Conclusion

Le site IHMN contient actuellement un nombre limité de documents PDF.

Ces documents sont principalement liés à :

- l'administration
- les inscriptions
- les obligations légales.

Une rationalisation des documents pourrait être envisagée lors d'une refonte du site, notamment en remplaçant certains documents informatifs par des pages web directement intégrées au site.
