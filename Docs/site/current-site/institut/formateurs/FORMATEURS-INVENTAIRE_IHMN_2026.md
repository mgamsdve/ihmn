# 3. Structure des fiches formateurs

Chaque formateur est présenté dans une fenêtre type **overlay / lightbox**.

Structure observée :

| Champ | Présence |
|------|------|
| Nom | toujours présent |
| Matières enseignées | toujours présent |
| Description / biographie | généralement présent |
| Lieu de consultation | parfois présent |
| Photo | présente sur la grille principale |

---

# 4. Structure technique 

La grille principale contient :

- photo
- nom
- ouverture d’une lightbox contenant la fiche complète

# 5. Navigation

La page fonctionne comme suit :

### grille principale

présentation des formateurs sous forme de portraits circulaires.

### ouverture des fiches

un clic ouvre une **fenêtre overlay** contenant :

- description

# 9. Limites et pistes d'amélioration

L’analyse de la page des formateurs met en évidence plusieurs éléments pouvant être améliorés dans la structure actuelle.

## Hétérogénéité des biographies

Les descriptions des formateurs ne suivent pas une structure strictement identique.

Certaines fiches contiennent :

- un parcours académique détaillé
- des informations sur leur pratique
- des lieux de consultation

D'autres sont plus succinctes et ne contiennent que quelques lignes de présentation.

Une structuration plus homogène des fiches pourrait améliorer la lisibilité globale.

## Informations de contact absentes

Contrairement aux fiches thérapeutes, les fiches formateurs ne contiennent généralement pas :

- téléphone
- email
- site web personnel

Cela limite les possibilités de contact direct avec les enseignants.

## Lien indirect avec les formations

Les formateurs sont mentionnés dans les modules de formation mais le site ne propose pas toujours de lien direct permettant :

- d'accéder à la fiche d'un formateur depuis un module
- d'identifier facilement tous les modules enseignés par un formateur.

Une meilleure interconnexion entre :

Formateurs  
→ Modules  
→ Formations

pourrait améliorer la compréhension du parcours pédagogique.

## Navigation limitée

La page fonctionne sous forme de grille avec ouverture de fiches via une fenêtre overlay.

Bien que cette présentation soit claire, elle présente certaines limites :

- impossibilité de partager un lien direct vers une fiche formateur
- difficulté à indexer individuellement les formateurs par les moteurs de recherche.

La création de pages individuelles pour chaque formateur pourrait améliorer :

- le référencement
- la navigation
- l’accessibilité des informations.

## Améliorations possibles dans une future refonte

Dans une future version du site, plusieurs améliorations pourraient être envisagées :

- pages individuelles pour chaque formateur
- structure uniforme des biographies
- liens directs entre formateurs et modules enseignés
- meilleure intégration dans la structure globale du site.

---

# 10. Conclusion

La page des formateurs remplit efficacement son rôle de présentation de l’équipe pédagogique de l’IHMN.

Elle met en avant l'expertise et la diversité des profils des enseignants, ce qui contribue à renforcer la crédibilité de l'institution.

Cependant, certaines améliorations structurelles pourraient permettre :

- une meilleure lisibilité
- une navigation plus fluide
- une interconnexion plus claire entre formateurs, modules et formations.

L’inventaire réalisé constitue une base de référence utile pour toute réflexion future concernant l’évolution de la structure pédagogique du site.
