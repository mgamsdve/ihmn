# Inventaire et analyse de la FAQ — IHMN (2026)

## Description

Analyse complète de la page FAQ du site IHMN afin de :

- recenser toutes les questions présentes
- analyser leur organisation
- catégoriser les questions par thème
- identifier les redondances éventuelles
- évaluer la qualité des réponses
- identifier les opportunités SEO
- proposer des recommandations pour une future restructuration.

URL analysée :

https://ihmn.be/faq

---

# 1. Structure générale de la FAQ

La FAQ est organisée en **grandes catégories thématiques**, chacune contenant plusieurs questions et réponses.

Les catégories identifiées sont :

- Naturopathie
- Notre école
- Nos formations, nos cours
- Inscriptions et dispenses
- Le minerval et les frais
- Après la formation

Cette organisation permet de couvrir l'ensemble du parcours d'un futur étudiant :

1. compréhension de la naturopathie  
2. présentation de l’école  
3. détails sur les formations  
4. processus d’inscription  
5. coûts  
6. débouchés professionnels

# 5. Redondances ou chevauchements

Quelques thèmes se recoupent :

### Organisation des cours

Plusieurs questions abordent :

- horaires
- organisation des journées
- absence à un cours.

Ces questions pourraient être regroupées dans une section :

**Organisation pratique des cours**

---

### Diplômes et certification

Les informations concernant :

- certificats
- examens
- stages
- travail de fin d’étude

pourraient être regroupées dans un thème :

**Certification et validation des études**

# 7. Potentiel SEO

La FAQ contient plusieurs questions correspondant à des **requêtes recherchées par les internautes**.

Exemples :

- qu'est-ce que la naturopathie
- formation naturopathie Belgique
- devenir naturopathe Belgique
- combien coûte une formation en naturopathie
- naturopathe reconnu Belgique

Ces questions possèdent un **fort potentiel SEO**.

Cependant :

- la FAQ n’est pas optimisée pour apparaître dans les résultats enrichis Google
- les réponses pourraient être structurées pour améliorer la lisibilité.

---

# 8. Analyse UX

Points positifs :

- navigation simple
- organisation par thème
- contenu complet

Limites :

- certaines réponses sont longues
- absence de recherche dans la FAQ
- navigation parfois longue sur mobile

Améliorations possibles :

- ajouter une table des matières
- réduire certaines réponses
- ajouter des liens vers les pages formations.
