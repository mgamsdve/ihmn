# Inventaire des pages institutionnelles

## Contexte

Objectif : recenser et analyser les pages institutionnelles du site actuel afin de préparer la migration vers la nouvelle architecture du site.

Les pages formations et l’annuaire des thérapeutes sont **exclus de cet inventaire**.

Le but est d’identifier :

- la structure du contenu
- la qualité rédactionnelle
- les éléments médias
- les opportunités de refonte ou suppression

Cette analyse servira de base pour :

- le **content model**
- la **nouvelle architecture du site**
- la **migration du contenu**

# Observations

## Points positifs

- contenu riche et détaillé
- FAQ très complète
- informations pratiques bien documentées
- crédibilité renforcée par les accréditations

## Problèmes identifiés

- architecture parfois confuse
- mélange entre pages institutionnelles et formations
- contenu parfois trop dense

## SEO observations

Aucune page institutionnelle du site ne possède de meta description définie dans le HTML.

Le CMS (Joomla) laisse donc Google générer automatiquement les extraits dans les résultats de recherche.

Cela représente une opportunité d'amélioration SEO lors de la refonte du site.

## Opportunités

- simplifier la structure des pages
- améliorer la hiérarchie des titres
- optimiser le référencement
- clarifier la proposition de valeur de l’école
