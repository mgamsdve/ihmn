# Inventaire complet de l'annuaire des thérapeutes — IHMN (2026)

## Description

Analyse complète de la structure et du contenu de l’annuaire des thérapeutes du site IHMN.

Objectifs :

- recenser les thérapeutes
- analyser la structure des fiches
- identifier les doublons
- analyser la navigation et la pagination
- documenter la carte interactive
- identifier les champs systématiques vs facultatifs

URL analysée :

https://ihmn.be/nos-therapeuthes

# 1. Structure générale de l'annuaire

L'annuaire des thérapeutes recense les praticiens formés à l'IHMN ayant obtenu un ou plusieurs des certificats suivants :

- Naturopathe
- Conseiller(ère) en hygiène de vie et nutrition
- Praticien(ne) en drainage lymphatique manuel

Chaque thérapeute possède une fiche comportant ses coordonnées et informations professionnelles.

# 12. Limites et pistes d'amélioration

L’analyse de l’annuaire des thérapeutes met en évidence plusieurs limites dans la structure actuelle du site.

## Hétérogénéité des fiches

Les fiches thérapeutes ne contiennent pas toujours les mêmes informations :

- certains thérapeutes possèdent un site web
- certains possèdent une adresse email
- certains disposent d'une description détaillée
- d'autres ne contiennent que des informations minimales

Cette hétérogénéité rend l'annuaire moins homogène et peut compliquer la recherche d'informations pour les visiteurs.

## Gestion des multi-localisations

Plusieurs thérapeutes apparaissent plusieurs fois dans l'annuaire car ils exercent dans plusieurs lieux de consultation.

Exemples :

- Marie VERWACHT
- Marie-Pierre BENOIT
- Nathalie SEGHERS
- Valérie GARAB
- Laurence MERTENS DE WILMARS

Ces entrées multiples peuvent être interprétées comme des doublons alors qu'elles correspondent à des localisations différentes.

Une amélioration possible serait de :

- créer une fiche unique par thérapeute
- associer plusieurs lieux de consultation à cette fiche.

## Absence de filtres avancés

L’annuaire permet uniquement une navigation par région.

Il n’existe pas de filtres permettant de rechercher les thérapeutes selon :

- les spécialités
- les certifications
- la distance géographique
- les types de consultations

L'ajout de filtres avancés améliorerait significativement l'expérience utilisateur.

## Complétude des données

Certains champs sont fréquemment absents :

- email
- site web
- description détaillée
- spécialités

Une normalisation des fiches pourrait améliorer la qualité globale de l'annuaire.

## Améliorations possibles dans une future refonte

Dans le cadre d'une refonte du site, plusieurs améliorations pourraient être envisagées :

- fiche thérapeute structurée avec champs obligatoires
- gestion des multi-localisations dans une même fiche
- filtres avancés (spécialité, type de pratique, région)
- moteur de recherche interne
- meilleure interconnexion avec les formations suivies par les praticiens.

---

# 13. Conclusion

L'annuaire des thérapeutes constitue une ressource importante pour l'IHMN, permettant de mettre en avant les praticiens formés par l'école.

La structure actuelle est globalement fonctionnelle mais présente certaines limites en termes de :

- cohérence des fiches
- gestion des multi-localisations
- possibilités de recherche et de filtrage.

L’inventaire réalisé permet de documenter précisément l’état actuel de l’annuaire et constitue une base solide pour :

- une refonte future
- une structuration améliorée des données
- une meilleure expérience utilisateur.
