# IHMN — Refonte du site web

## Contexte

Le site actuel de l’Institut d’Hygiène et de Médecine Naturelle (IHMN) est développé sous Joomla et contient de nombreuses informations sur les formations, l’école et les thérapeutes formés par l’institut.

Le projet consiste à reconstruire ce site avec une architecture moderne, afin d'améliorer :

- la lisibilité des contenus
- la navigation
- le référencement naturel
- la gestion du contenu par l’équipe de l’IHMN

---

# Objectifs

- moderniser l’interface et la structure du site
- améliorer le référencement naturel (SEO)
- clarifier la présentation des formations
- améliorer le parcours des visiteurs
- permettre à l’équipe de modifier facilement le contenu

---

# Architecture technique

## Frontend

Application web moderne optimisée pour :

- performance
- SEO
- accessibilité
- responsive design

## Gestion de contenu

Le contenu sera géré via un **Headless CMS (Sanity)**.

Le CMS permettra de gérer :

- pages du site
- formations
- modules de cours
- formateurs
- thérapeutes
- articles de blog
- FAQ

---

# Phase 1 — Structure du site

Définition de la structure du site et de l’organisation du contenu.

Travail réalisé :

- analyse du site actuel
- définition de l’architecture du futur site
- structuration des pages principales
- identification des opportunités SEO

## Livrables

- arborescence du site
- structure des pages
- organisation des contenus
- liste de sujets d’articles pour le blog

---

# Phase 2 — Développement

Création du nouveau site web.

Fonctionnalités principales :

- pages de présentation de l’école
- pages de formations
- programme structuré par années et modules
- pages formateurs
- section thérapeutes avec carte et filtres
- FAQ
- blog
- page contact

---

# SEO

Le site intégrera les bases du référencement naturel :

- titres de pages optimisés
- meta descriptions
- structure H1 / H2 / H3
- URLs propres
- maillage interne
- données structurées (schema.org)
- sitemap

Le blog permettra de publier du contenu autour de :

- naturopathie
- hygiène de vie
- nutrition
- bien-être
- métier de naturopathe
