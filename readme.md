# IHMN - Projet de refonte du site web

## Objectif

Ce depot centralise la preparation de la refonte du site IHMN (https://ihmn.be).

Le projet sert a:

- documenter le site actuel
- organiser les contenus a migrer
- preparer la future architecture du site
- garder une base de reference claire pour la suite

## Etat du projet (mars 2026)

Avancement confirme:

- EPIC 1 termine: inventaire complet du contenu actuel
- EPIC 2 termine: analyse de l'architecture actuelle (navigation, URLs, maillage, CMS)

Prochaines etapes:

- EPIC 3: audit UX
- EPIC 4: audit SEO
- EPIC 5 a 8: synthese, architecture cible, plan de migration et rapport final

## Contenu du repository

- `Docs/project/`: cadrage projet (decisions, roadmap, notes)
- `Docs/site/current-site/`: inventaires detailes du site existant
- `Docs/site/`: analyses transverses (architecture, SEO, contenu)
- `scripts/`: scripts utilitaires d'extraction et de consolidation
- `web-scrap/`: archive HTML du site actuel pour travail hors ligne

## Perimetre couvert

- pages institutionnelles de l'ecole
- pages formations
- annuaire des therapeutes
- FAQ, contact, documents PDF
- base de travail pour la future arborescence du site
